Mouda is a startup in Korea, which provides peer-to-peer lending platform.
For 3 months, I worked as web design intern at Mouda.
The main task was to design a responsive web site from scratch.

During the internship, I spnet most of my time on designing the main page.
At first, I designed the main page with pictures and few texts so that
it catches the users eyes first, and then explains about P2P lending and the company.

However, after the closed beta testing, we realized that
some users including those old elders who are not familiar with
such website need a guide for using the website.

So, I created tutorial videos and added them into the main page.
The users can choose the guide video from the tab which says
"Registration Guide", "Investing guide" and "Lending Guide"
The users were satisfied with the tutorial videos.

Although the current webisite has changed,
the layout is still the same as I designed.
